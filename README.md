# Analog Clock

A simple **Analog Clock** built using **HTML, CSS, and JavaScript**.  
This project demonstrates how to create a functional clock with smooth hands movement using web technologies.

---

## 🚀 Features

- Real-time analog clock
- Smooth second, minute, and hour hand movement
- Responsive design
- Pure HTML, CSS, and JavaScript (no external libraries)
